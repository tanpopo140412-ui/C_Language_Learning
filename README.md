# C_Language_Learning
。
C言語の基礎学習・カレンダー作成

Try1.cファイル : ポインタを使用したカレンダーづくりのための試作その１。元は数か月前に制作したもの。
年月日を入力すると前日・次の日の年月日が表示されるようになっているはず。自分なりに考え書いたコードの中で特に長めで、間違った個所を見つけるのも訂正するのも力を要した。
Git Hubに初めてリポジトリとして公開する練習も兼ねている。

Try2.cファイル : ポインタを使用したカレンダーづくりのための試作その２。
Try1.cファイルの致命的な論理破綻や冗長さを関数を複数に分けることや、セルフデバッグ・外部からのフィードバック（AIチューター）などにより時間をかけて段階的に改善したもの。
西暦１年未満の場合などはエラー表示を出し、また世界標準に合わせているため暦が変わる１５８２年以降・以前は区別しない。今はあくまで日常的なものを作成することにしている。

Try3.cファイル : カレンダー試作のためのツェラーの公式を使用した曜日を表示するためのコード。処理系によって変化する可能性があるため注意。


Calender1.cファイル : 構造体を使用したカレンダーの試作。Try2.cファイルの改良版。表示するだけであるため、不要なポインタ部分を修正した。
